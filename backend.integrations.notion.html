backend.integrations.notion module

async backend.integrations.notion.authorize_notion(user_id, org_id)[source]

Initiates the Notion OAuth2 authorization flow.

Generates a unique state, stores it in Redis, and returns the Notion authorization URL.

Parameters:

  • user_id (str) – The ID of the user initiating the authorization.

  • org_id (str) – The ID of the organization.

Returns:

The Notion authorization URL.

Return type:

str

async backend.integrations.notion.oauth2callback_notion(request: Request)[source]

Handles the OAuth2 callback from Notion.

Validates the state, exchanges the authorization code for an access token, and stores the credentials in Redis.

Parameters:

request (fastapi.Request) – The FastAPI request object containing query parameters.

Raises:

HTTPException – If an OAuth error occurs, state mismatch, or token exchange fails.

Returns:

An HTML response to close the browser window.

Return type:

fastapi.responses.HTMLResponse

async backend.integrations.notion.get_notion_credentials(user_id, org_id)[source]

Retrieves Notion credentials for a given user and organization from Redis.

Deletes the credentials from Redis after retrieval.

Parameters:

  • user_id (str) – The ID of the user.

  • org_id (str) – The ID of the organization.

Raises:

HTTPException – If no credentials are found.

Returns:

A dictionary containing the Notion credentials.

Return type:

dict

async backend.integrations.notion.create_integration_item_metadata_object(response_json: dict) → IntegrationItem[source]

Creates an IntegrationItem object from Notion response data.

Extracts relevant properties from a Notion page or database item to populate an IntegrationItem instance. Skips rows if essential fields are missing.

Parameters:

response_json (dict) – The JSON response data from the Notion API for a page or item.

Returns:

An IntegrationItem object representing the Notion item, or None if validation fails.

Return type:

Optional[IntegrationItem]

async backend.integrations.notion.get_items_notion(credentials) → list[dict][source]

Fetches Notion items (pages/databases) for a given set of credentials.

Checks for cached items in Redis. If not found, fetches from Notion API, creates IntegrationItem objects, and caches the results.

Parameters:

credentials (str) – A JSON string containing the Notion access token.

Raises:

HTTPException – If fetching items fails or an unexpected error occurs.

Returns:

A list of dictionaries, each representing a Notion item.

Return type:

list[dict]
